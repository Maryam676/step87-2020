first in, first out: virtual office hours made easy



As courses have moved to a virtual setting, existing office hours management systems fall short in providing the same level of seamless interaction with students.

The site is geared towards teaching staff of computer science courses to use with students during virtual office hours as a better alternative to screen-sharing in Zoom. The website allows teaching staff to create shared workspaces that allows users to edit code and collaborate in real time.
